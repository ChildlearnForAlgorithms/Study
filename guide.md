### updateTestcase 이용방법
- 파일로 불러오기(0)  
입력값 : 문제 번호
update.txt로 testcase를 가져옴
  
  
- 파일 업데이트(1)  
update.txt 이용  
  (맨 끝 엔터 필수)
<pre><code>problem [문제번호]
{반복}
  input[번호]
  [input value]

  output[번호]
  [output value]

</code></pre>
  

- 테스트케이스 삭제(2)  
입력값 : 문제 번호, 테스트케이스 입력


- 문제 삭제(3)  
입력값 : 문제 번호

- 테스트케이스 출력 (4)  
입력값 : 문제 번호
